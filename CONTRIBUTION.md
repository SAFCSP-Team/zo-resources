# Contributing to Bootcamp Resources

Thank you for your interest in contributing! We welcome all forms of contributions that help improve our learning resources.

## How to Contribute

### Reporting Issues
Found a typo, error, or have a suggestion?
1. Go to the [Issues](../../issues) tab
2. Click "New Issue"
3. Describe the problem clearly with examples if possible
4. Submit and we'll review it

### Submitting Changes
Want to fix something or add content?
1. Fork the repository
2. Create a new branch: `git checkout -b fix/your-change`
3. Make your changes
4. Commit with a clear message: `git commit -m "Fix: description of change"`
5. Push to your branch: `git push origin fix/your-change`
6. Open a Pull Request describing your changes

## Guidelines

- Keep content clear and beginner-friendly
- Follow existing formatting and style
- Test code examples before submitting
- Provide context and explanations
- Be respectful and constructive
